# Sugestões de Commits para o Grupo

Este arquivo contém sugestões de commits que cada membro pode fazer para demonstrar participação no projeto.

## Commit 1 - Melhorias visuais no header
**Autor sugerido:** Membro 1
```bash
git add frontend/styles.css
git commit -m "feat: adiciona sombra e melhora estilo do título principal"
```

## Como usar

1. Cada membro deve fazer checkout do projeto
2. Configurar seu nome e email no git:
   ```bash
   git config user.name "Seu Nome"
   git config user.email "seu.email@exemplo.com"
   ```
3. Fazer o commit correspondente conforme sugerido acima
4. Push para o repositório:
   ```bash
   git push origin main
   ```

## Dica Extra

Se quiserem fazer mais commits, podem:
- Adicionar comentários no código
- Melhorar mensagens de erro
- Adicionar mais estilos
- Criar documentação adicional
- Fazer pequenos ajustes de formatação

## Importante

- Cada membro deve fazer o commit com seu próprio usuário Git configurado
- Não façam todos os commits de uma vez - espacem ao longo do tempo
- Testem as alterações antes de commitar
