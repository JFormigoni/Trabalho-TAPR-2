# Frontend E-Commerce

Frontend básico para demonstração do sistema de microserviços.

## Como Executar

1. Certifique-se de que todos os serviços backend estão rodando
2. Abra o arquivo `index.html` em um navegador web

## Funcionalidades

- ✅ Login e Registro de usuários
- ✅ Listagem de produtos
- ✅ Adicionar produtos ao carrinho
- ✅ Visualizar carrinho
- ✅ Finalizar pedido
- ✅ Visualizar histórico de pedidos

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Configuração

O frontend está configurado para se conectar ao gateway em `http://localhost:8083`.
Se necessário, altere a constante `API_BASE` no arquivo `app.js`.
